How to Check Your Memory

cat /proc/meminfo

How to Check Your Storage

To Find your drives 
lsblk
To see disk usage 
df -h /dev/sda1

How to Update Your System

This assumes you are running ARCH Linux 
sudo pacman -Syu --noconfirm

How To Update ASIMOS

This changes to the default directory location, pulls the latest version and then installs any new node dependancies

cd ~/asimos/ && git pull && npm install

How to Reboot Your Device

This assumes you have bitcoin running and don't want to corrupt your database

bitcoin-cli stop && sudo reboot

How to Shutdown Your Device

This assumes you have bitcoin running and don't want to corrupt your database

bitcoin-cli stop && sudo shutdown -t now 0
